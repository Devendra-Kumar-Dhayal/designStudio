

## Quick start

To setup using docker compose

```bash
  docker compose up --build --no-cache
```
to setup manually 
naviage to client folder;
```bash
  yarn 
  #or
  npm install --force
  npm start
  
```
naviageto server folder;
```bash
  yarn 
  #or
  npm install --force
  npm run dev
  
```
setup .env file using .env.example
at client side add .env.local
and for server side .env
